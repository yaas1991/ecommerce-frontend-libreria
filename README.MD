# E-commerce Frontend: Librería Cósmica

## Descripción

Este es un MVP de una tienda en línea para una librería,
libreria ficticia con tematica fantastica

## Objetivos Cumplidos

- Estructura con HTML5 semántico.
- Diseño responsivo con Bootstrap.
- Interactividad básica con JavaScript (agregar al carrito, etc.).
- Gestión con Git/GitHub.

## Instalación y Ejecución

1. Clona el repositorio: `git clone https://github.com/tu-usuario/tu-repo.git`
2. Abre `index.html` en un navegador.

## Enlace al Repositorio

https://github.com/tu-usuario/tu-repo (reemplaza con tu URL real)
